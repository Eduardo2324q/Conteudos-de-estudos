🌐 HTML Básico

HTML é a linguagem usada para estruturar o conteúdo de páginas na web. Ela usa tags para marcar elementos como títulos, parágrafos, links e imagens.

🎨 CSS Básico

CSS é a linguagem usada para estilizar páginas na web. Com ela, você define cores, fontes, tamanhos e posicionamento dos elementos. As regras CSS são aplicadas usando seletores, propriedades e valores.
