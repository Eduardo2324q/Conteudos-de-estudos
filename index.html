🎨 CSS Básico

CSS é a linguagem usada para estilizar páginas na web. Com ela, você define cores, fontes, tamanhos e posicionamento dos elementos. As regras CSS são aplicadas usando seletores, propriedades e valores.

🎨 CSS Avançado - Flexbox

Flexbox é uma técnica CSS avançada que facilita o layout flexível e responsivo, organizando itens em linhas ou colunas. Ele permite controlar alinhamento, espaçamento e distribuição dos elementos dentro de um contêiner, adaptando-se dinamicamente ao tamanho da tela. Ideal para criar designs que se ajustam facilmente a diferentes dispositivos.

🔗Ver no Notion

🎨 CSS Avançado - Grid

CSS Grid é um sistema avançado de layout que organiza conteúdo em linhas e colunas, formando uma grade bidimensional. Ele oferece controle preciso sobre o posicionamento e o tamanho dos elementos, facilitando a criação de layouts complexos e responsivos de forma intuitiva. É perfeito para estruturar páginas com múltiplas áreas alinhadas tanto horizontal quanto verticalmente.
